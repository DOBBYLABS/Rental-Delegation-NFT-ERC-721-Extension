# EIP-X

EIP-X is a rental and delegation extension for EIP-721. It allows owners to lend NFTs without the necessary staking mechanism. The NFT retains original user while user is added.

## Install
```shell
npm install
```

## Compile contracts:
```shell
npx hardhat compile
```

## Run tests:
```shell
npx hardhat test
```
